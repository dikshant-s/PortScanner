About Port Scanner

The Port Scanner is a web application designed to help users identify open ports on a remote host. It allows users to input a hostname and a range of ports to scan, providing results on which ports are open.

This tool is particularly useful for network administrators and security professionals to assess the security of their systems and detect potential vulnerabilities.

Features

  • Scan multiple ports in a specified range.
  • Display the status of each port (Open/Closed).
  • Easy-to-use interface with real-time feedback.

Technologies Used

The application is built using HTML, CSS (Materialize framework), JavaScript, and a backend server powered by Node.js and Express.
